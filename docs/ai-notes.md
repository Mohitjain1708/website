# AI Notes

AI was used as a build accelerator, not as a replacement for product thinking.

## Used For

- selecting a high-signal property narrative
- refining the commercial storyline for tenants, sponsors, and event partners
- tightening the page structure into clear chapters
- drafting the README, workflow notes, and ATS bullets

## Not Used For

- no generated code was copied without adaptation
- no hidden automation or black-box UI patterning
- no attempt to imitate a generic AI template look

## Visual Strategy

The implementation leans on bespoke SVG posters and motion-friendly surfaces so the deck feels authored and premium while staying lightweight.
